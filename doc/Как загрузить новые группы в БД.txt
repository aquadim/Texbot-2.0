Зато можно загрузить id семестров из АВЕРС. Для этого нужно просто выполнить файл load-periods.php, передав как аргумент файл с id семестров (см. описание в doc/).

Скрипт может выдавать предупреждения о том что строка не обработана. Это значит, что в строке указана группа со специальностью и годом поступления, которой нет в базе данных.
